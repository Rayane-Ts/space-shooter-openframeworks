# Space Shooter (openFrameworks)

Made by Rayane Tsouli & Yassine Amraoui

A small 3D space-shooter made with openFrameworks:
- Fly a spaceship.
- Asteroid field with rotation
- Shoot projectiles and destroy asteroids
- 3 camera views: main third-person + 2 mini viewports (top-down (acts like a minimap kind of) + cockpit(first person) )
- Optional sound effects (engine loop, laser, explosion)
- Toggle grid

---

 Controls

- W / S: forward / backward throttle
- A / D: strafe left / right
- Q / E: up / down
- Mouse drag: rotate ship
- Space: fire projectile
- R: reset world
- G: toggle grid

HUD shows asteroid count, projectile count, speed, and grid status.

---

## Required Setup

This project uses:
- openFrameworks (Windows / macOS / Linux)
- ofxAssimpModelLoader (ships with OF)
